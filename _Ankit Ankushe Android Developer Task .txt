Android Developer Task
Duration - 24 Hours

🔹 Objective
Design and develop a scalable Android application using modern architecture and best practices. The app will display a list of products fetched from a public API, with filtering, detailed views, and dynamic UI handling.


🔹 Tech Stack Requirements
Android Jetopack
Architecture: MVVM (Model-View-ViewModel)
Dependency Injection: Hilt
Networking: Retrofit
Asynchronous Programming: Kotlin Coroutines / Flow
UI Components: RecyclerView, ViewBinding / Jetpack Compose (optional but preferred)
Image Loading: Coil
State Management: LiveData / StateFlow


🔹 API Details
Product List API: https://api.freeapi.app/api/v1/public/randomproducts?page=1&limit=100
Product Detail API: https://api.freeapi.app/api/v1/public/randomproducts/{id}
